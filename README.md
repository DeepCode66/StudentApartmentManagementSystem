## 功能介绍（完整见运行截图）
管理员：基本功能包括登录、注册和退出。可以访问控制台、宿舍楼管理、学生信息、缺勤记录和系统设置模块。能够新增、编辑和删除宿舍楼及缺勤记录信息，查看详细记录，确保系统正常运行和信息维护。

学生：主要功能是登录和注册。可以查看个人缺勤记录及相关学生信息。在注册时选择对应的注册类型以确定用户权限，并能够查询和管理自身的缺勤记录信息，保障个人信息的及时性和准确性。
